Fault Trees App

Software para Modelação de Árvores de Falhas

Sobre o Projeto

No contexto da engenharia de sistemas e fiabilidade, o desenho de Árvores de Falhas (Fault Tree Analysis - FTA) é crucial para prever e mitigar falhas num sistema complexo. Este projeto consistiu na criação de uma aplicação dedicada para desenhar e estruturar estas árvores de forma intuitiva.

A aplicação permite mapear a relação lógica entre falhas de componentes (eventos básicos) e a falha de todo o sistema (evento de topo), utilizando portas lógicas booleanas (AND, OR), otimizando o processo de análise de risco.
